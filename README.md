# HowWasToday? (你今天过的怎么样？)

一个陪伴你、倾听你的日常，并自动帮你记录生活和生成日记的桌面端AI小助手。让生活中的每一次灵光一现和小确幸都能被温柔接住。

## ⚙️ 核心功能
* 💬 **轻飘陪伴**：以置顶悬浮小窗口常驻桌面，无需专门打开工具，随时随地想聊就聊。
* 🧠 **连贯记忆**：自动提取你的喜好和状态（短/长期记忆分层），提供更懂你的专属对话，不会“每次重新认识”。
* 🙋 **主动机制**：在你空闲时或特定的作息时间点，会像朋友一样出现，主动开口关心“你今天过得怎么样”。
* 📖 **自动日记**：根据当天的聊天碎片，自动合并、总结和提炼情绪，生成一篇属于你并且可以回顾的心情日记。
* 🌐 **联网检索**：支持网络查询最新信息，紧贴现实与你交流。

## ⬇️ 怎么下载
本软件完全开源且免配置即可安装体验。
1. 在本项目仓库中，找到 **`releases`** 文件夹。
2. 下载其中的安装包 `LifePartnerAI Setup 1.0.0.exe`。
3. 双击下载好的 `.exe` 软件，根据提示完成安装（目前支持 Windows 系统）。

## 🚀 怎么使用
安装并在电脑上运行软件后，建议进入软件内置的**设置**页面（齿轮图标 ⚙️），进行一下简单的配置：

* **方式一：使用测试版**
  - 应用内置了默认的一些测试配置，不需折腾就可以直接与你对话（此方法适合快速验证功能，但公共测试 API 可能会有额度或响应限制）。
  
* **方式二：自己输入 API 密钥（强推⭐）**
  1. 为了获得极速、私密且连续的陪伴体验，建议填入你个人专属的 API Key。
  2. 前往提供大型模型服务的平台（如硅基流动 SiliconFlow 免费获取，或者填写 OpenAI 格式的智谱、DeepSeek 等渠道密钥均可）。
  3. 在设置中将你申请的 **`API Key`** 和对应的 **`模型名称(Model)`** 填入保存。
  4. 回到主界面，开始与“TA”分享你一天的生活！
